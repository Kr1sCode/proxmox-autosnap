konfiguracja pierwszego uruchomienia
1 · Utwórz token API na hoście Proxmox
Pokaż komendy (uruchom na hoście PVE jako root)
Nadaje tokenowi tylko prawa do migawek i odczytu maszyn — nic więcej. 
pveum role add AutoSnap --privs "VM.Snapshot VM.Audit"
pveum user add autosnap@pve
pveum acl modify /vms --users autosnap@pve --roles AutoSnap
pveum user token add autosnap@pve manager --privsep 0
Ostatnia komenda wypisze value — wklej poniżej pełny token w formacie autosnap@pve!manager=<value>.
