Show commands (run on the PVE host as root)
Grants the token only snapshot + read rights — nothing else. 
pveum role add AutoSnap --privs "VM.Snapshot VM.Audit"
pveum user add autosnap@pve
pveum acl modify /vms --users autosnap@pve --roles AutoSnap
pveum user token add autosnap@pve manager --privsep 0
The last command prints a value — paste the full token below as autosnap@pve!manager=<value>.
